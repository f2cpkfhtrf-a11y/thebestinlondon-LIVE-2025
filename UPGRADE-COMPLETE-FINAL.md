## 🔑 Get Google Places API Key (OPTIONAL)

The site works perfectly WITHOUT an API key (using beautiful gradients).

But if you want REAL restaurant photos:

1. Go to: https://console.cloud.google.com/apis/credentials
2. Create a new project (or select existing)
3. Enable "Places API"
4. Create credentials → API key
5. Copy your key
6. Open `.env.local` file
7. Paste key after `NEXT_PUBLIC_GOOGLE_PLACES_KEY=`

**Example:**
```
NEXT_PUBLIC_GOOGLE_PLACES_KEY=AIzaSyC1234567890abcdefghijklmnop
```

8. Restart dev server: `npm run dev`
